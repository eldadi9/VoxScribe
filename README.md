# VoxScribe

AI Transcription and Meeting Summaries

## Overview
VoxScribe is a SaaS product for recording, transcribing, and summarizing meetings, conversations, and media files.

The platform is planned to support:
- live recording and transcription
- audio and video file upload
- multilingual transcription
- speaker separation
- timestamps
- AI summaries
- action items
- workflow summaries
- export to text, PDF, and SRT
- future integrations with meeting and productivity tools

## Current Project Status
This project is currently in the planning and setup phase.

At this stage, the repository includes:
- project folder structure
- planning documents
- backlog files
- status tracking files

The application logic is not implemented yet.

## Project Structure
- `docs/` - product and technical planning documents
- `backlog/` - MVP and phased backlog
- `status/` - project tracking and progress files
- `backend/` - future FastAPI backend
- `frontend/` - future frontend app
- `workers/` - future background processing services
- `infra/` - future infrastructure configuration
- `scripts/` - helper scripts

## Product Direction
VoxScribe is being designed as a modular platform for:
1. recording and transcribing conversations
2. uploading and transcribing media files
3. generating AI summaries and action items
4. exporting structured outputs
5. connecting to external tools in later phases

## Tech Direction
Planned stack:
- Backend: FastAPI + Python
- Frontend: Node.js + TypeScript
- Database: PostgreSQL
- Media processing: FFmpeg
- Workers: background processing pipeline

## Notes
This repository is being built step by step.
No major implementation should begin before the planning documents are completed and approved.
